# Neynar API Features to Democratize

Based on Neynar API documentation, here are all the features we'll make available through @yeynar mentions:

## Core Categories

### 1. **User** (16 endpoints)
- Get user profiles and data
- Search users
- Bulk user lookups
- Power users discovery
- User activity and stats

### 2. **Cast** (17 endpoints)
- Fetch casts and threads
- Search casts
- Post new casts
- Reply to casts
- Delete casts
- Cast analytics

### 3. **Signer** (18 endpoints)
- Manage signers for posting
- Create and revoke signers
- Signer permissions

### 4. **Auth Address** (19 endpoints)
- Manage authentication
- Linked addresses

### 5. **Feed** (20 endpoints)
- User feeds
- Channel feeds
- Trending feeds
- Following feed
- Custom feeds

### 6. **Reaction** (21 endpoints)
- Like/unlike casts
- Recast functionality
- Get reactions

### 7. **Follow** (22 endpoints)
- Follow/unfollow users
- Get followers/following
- Follow suggestions

### 8. **Mini Apps** (23 endpoints)
- Farcaster Frames
- Frame validation
- Frame analytics

### 9. **Mini App Hosts** (24 endpoints)
- Host frames
- Frame deployment

### 10. **Onchain** (25 endpoints)
- Onchain data integration
- Token balances
- NFT ownership

### 11. **Agents** (26 endpoints)
- AI agent creation
- Agent interactions
- Automated responses

### 12. **Notifications** (27 endpoints)
- Push notifications
- Notification management
- Real-time alerts

### 13. **Storage** (28 endpoints)
- Data storage
- File management

### 14. **Channel** (28 endpoints)
- Channel discovery
- Channel feeds
- Channel membership
- Channel analytics

## Yeynar Bot Commands (User-Friendly)

Users can mention @yeynar with natural language queries like:

- "Show me stats for @username"
- "What's trending?"
- "Who should I follow?"
- "Search for casts about crypto"
- "Show me channel analytics for /yeynar"
- "What are my notifications?"
- "Find users interested in AI"
- "Show me top casters today"

The bot will use Gemini AI to understand the intent and call the appropriate Neynar API endpoints.
